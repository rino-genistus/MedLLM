# MedLLM

MedLLM is a Retrieval-Augmented Generation (RAG) agent designed to help medical students study more efficiently. Instead of searching through dense notes, textbooks, and lecture material manually, users can simply ask a question and receive a clear, source-grounded answer in seconds.

Built with React, Vite, FastAPI, and Python, MedLLM combines a responsive frontend with a powerful backend pipeline for semantic retrieval and natural-language quesetion answering. The goal is to make studying faster, smarter, and more interactive by turning complex medical content into accessible explanations.
